| 对比项 | Flutter | React Native |
|--------|---------|--------------|
| 开发语言 | Dart | JavaScript / TypeScript |
| 渲染引擎 | Skia（自绘引擎） | 原生桥接（JSI / Fabric） |

| 当前最新稳定版 | 3.x | 0.76+（新架构） |
